Carleton Video Downloader

This Chrome extension allows you to download video lectures from Brightspace at Carleton University.

How it works:
  • Navigate to a Brightspace lecture page.
  • Click the "Sniff" button added to the page.
  • If not visable reload page.
  • play the video/Lecture you want to download
  • A "Download" button will appear next to the video.
  • Click the "Download" button to save the lecture.
  • If any issues occur, please contact on github
